This repository contains the implementation of image classification with Arduino Nano 33 BLE Sense board. 
This project is based on the Edge Impulse framework for Arduino: https://github.com/edgeimpulse/firmware-arduino-nano-33-ble-sense/tree/master/src

For the report, please refer to "Image_Classification_Report.pdf". It is accompanied by "Project_Notes.pdf" which contains related instructions and problem solutions.
